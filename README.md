# Projeto de Engenharia de Software 2020.3 - UFRPE
## Gerenciador de Despesas Gerais

Esta aplicação, visa gerenciar todas as despesas do usuário, onde será possível configurar e relacionar as receitas, alertando caso as despesas superem uma determinada porcentagem da receita (essa porcentagem será configurada pelo usuário). Será possível também, visualizar gráficos com a relação de receita e despesas mensais e anual.
